CEMIC DIGITAL - Protótipo

Este site foi desenvolvido como protótipo para um acervo digital histórico-cultural,
financiado com recursos da PNAB 2025.

✔ Site estático em HTML/CSS/JS
✔ Estrutura simples e acessível
✔ Pode ser hospedado gratuitamente no GitHub Pages

Páginas incluídas:
- index.html (Início)
- sobre.html (Apresentação)
- acervo.html
- publicacoes.html
- mediacao.html
- contato.html

Para rodar localmente:
1. Abra esta pasta no VS Code
2. Clique com o botão direito em index.html e selecione "Open with Live Server"

Este projeto é uma sugestão e pode ser expandido ou institucionalizado pela administração pública.
